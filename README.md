# Rock Paper Scissors Game 🎮

A fully interactive, console-based Rock, Paper, Scissors game written in C++. This project was built with a strong focus on Clean Code principles, demonstrating modular architecture, efficient data management, and a robust game loop.

## ✨ Features
* **Customizable Matches:** Choose to play anywhere from 1 to 10 rounds per game.
* **Dynamic Score Tracking:** Accurately calculates and tracks Player wins, Computer wins, and Draws throughout the game.
* **Visual Terminal Feedback:** Utilizes system commands to dynamically change terminal colors based on round results (Green for Player win, Red for Computer win, Yellow for Draw).
* **Infinite Replayability:** Built-in safe loop allowing users to reset and play again without experiencing memory leaks or stack overflows.
* **Clean Architecture:** Code is heavily structured using `Enums` and `Structs` for data modeling, alongside single-responsibility functions.

## 🛠️ Technologies Used
* **Language:** C++
* **Concepts:** Control Flow, Structs, Enums, Random Number Generation, Standard I/O.
